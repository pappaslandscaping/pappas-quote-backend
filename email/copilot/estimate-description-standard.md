## Style Rules

- Use `Helvetica, Arial, sans-serif` only.
- Keep the `What's Included` label identical across services.
- Always use the accent rule: `28px x 2px` in `#c9dd80`.
- Use the summary paragraph first at `14px` with `1.7` line-height.
- Use a left divider `3px solid #d7dfd1` for the detail section.
- Keep detail lines at `13px` with bold labels in `#2e403d`.
- Use `#6e7f6d` only for the muted optional line.
- Keep the layout compact: one summary paragraph, 2 to 3 detail lines, and one optional note if needed.
